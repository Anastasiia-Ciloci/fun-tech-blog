## Installation

Step 1. Install npm

```
npm init -y
```

Step2.Install all modules you need

```
npm install express sequelize mysql dotenv bcrypt express-session
```

Step3. Create a table in sql

```
mysql -u root

SOURCE db/schema.sql;

exit
```

Step4.Nodemon for connecting to a server

```
nodemon

```
